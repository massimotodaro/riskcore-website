## Visual Reference Instructions

To create your visual reference PDF:

1. Open http://localhost:3000 in your browser
2. Click each link above to scroll to that section
3. Take a screenshot of each section (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
4. Paste screenshots into a document with the section names

Or use browser print to PDF:
1. Visit http://localhost:3000
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select "Save as PDF"
4. This will capture the full page
